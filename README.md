# FlightRoute

FlightRoute 是一个**监控流量去向的可视化地图模块**。

它的目标不是绑定某一个面板或某一个宿主，而是提供一套可复用的：
- 世界地图资产
- 经纬度到 SVG 坐标映射
- 流量点位与飞线建模
- 宿主适配接口

这样不同来源的数据都可以接入这块面板，例如：
- 网络面板
- 路由器设备
- 自定义流量采集器
- 其他宿主应用

## 设计原则
- **独立内核**：核心渲染和建模逻辑不依赖具体宿主
- **宿主注入**：宿主通过 adapter 提供 IP、Geo、连接数据等输入
- **资产可复用**：SVG、标定数据、生成脚本尽量保持宿主无关
- **多宿主接入**：同一块地图面板可接入不同数据来源

## 当前目录
- `assets/`：地图资产、源数据、生成结果、生成脚本
- `render/`：渲染内核与流量建模逻辑
- `adapters/`：宿主适配说明与草稿
- `docs/`：架构、状态、策略文档
- `notes/`：迁移计划与临时笔记
